🔑 Obtener Refresh Token

Usá esta página una sola vez para obtener el token de Microsoft y guardarlo en Vercel.

Paso 1 — Iniciar sesión con Microsoft

Hacé click para autenticarte con tu cuenta de OneDrive donde tenés el Excel.

✅ Paso 2 — Copiá este Refresh Token

Guardá este valor como variable de entorno MS_REFRESH_TOKEN en Vercel:

Cargando…
Paso 3 — Variables de entorno en Vercel:
  1. Abrí vercel.com → tu proyecto → Settings → Environment Variables
  2. Agregá: MS_CLIENT_ID = 7c0a9ce5-ed5f-4192-8a38-774d08ffad25
  3. Agregá: MS_REFRESH_TOKEN = el token que copiaste arriba
  4. Hacé Redeploy
⚠️ Borrá este archivo (gettoken.html y api/gettoken.js) del repo una vez que guardaste el token en Vercel.

🔧 ¿Ya tenés un token? Guardalo directo en el Blob

Si ya obtuviste un refresh token (por ejemplo desde Vercel env vars o desde otra sesión), pegalo acá y lo guardamos en el Blob sin necesidad de volver a iniciar sesión.
